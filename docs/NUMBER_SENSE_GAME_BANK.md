# 数感游戏题库

说明：本题库面向幼小衔接儿童，题面按可视化游戏改写，可直接用于 `/learn/tasks/math` 的题目数据。题目借鉴公开早期数学资源中的活动框架，如点卡、10 框、数线、隐藏数量、生活故事，但题面和参数为原创整理。

推荐数据字段：

```ts
type NumberSenseQuestion = {
  id: string;
  module: "quantity_compare" | "dot_flash" | "ten_frame" | "number_line" | "compose_decompose" | "story_problem";
  prompt: string;
  display: string;
  answer: string;
  hint: string;
  difficulty: 1 | 2 | 3 | 4;
};
```

## 使用建议

- 低龄或刚开始学习：先使用 NS-001 到 NS-060，重点让孩子看见、摆动、配对数量。
- 已能稳定数到 10：加入 NS-061 到 NS-100，强化数线、前后关系、组成分解。
- 幼小衔接后段：加入 NS-101 到 NS-120，把数量直觉迁移到生活故事。
- 游戏里不要先展示算式。先展示圆片、点卡、10 框、数线或实物，再在反馈阶段出现数字和算式。
- 提示应分层：先让孩子观察，再引导配对或补满 5/10，最后才给出数字关系。
